Information about Test Console

Test Console Portal provides a centralized control to execute Automation Tests using multiple Test Tools available in Verizon.

This Portal will be helpful for both Management and Testers, as it provides options to execute Test Cases and Display test reports.

Announcements

The Test console portal is currently integrated end to end with Revo automation tool.

In the next subsequent releases, the plan is to include other tools like Appium and Storm, under the console portal.
